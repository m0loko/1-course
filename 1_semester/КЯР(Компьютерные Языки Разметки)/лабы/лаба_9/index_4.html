С помощью свойства display: grid устанавливается контейнер, который будет использовать grid-верстку. Этот контейнер становится "сеткой", и все его дочерние элементы (называемые "сетки") будут размещаться в этой сетке.
Эти свойства позволяют задавать размеры столбцов и строк в сетке. grid-template-columns определяет, как будут распределяться столбцы
Это свойство задает промежутки между ячейками сетки. Оно работает как для столбцов, так и для строк. Можно использовать одно свойство для установки одинаковых отступов между всеми ячейками или указать разные значения для столбцов и строк.
С помощью этого свойства можно определить, в каких колонках должна располагаться ячейка. Это позволяет управлять расположением элементов в сетке, например, объединять несколько ячеек.
